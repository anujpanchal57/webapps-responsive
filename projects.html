MHGen Dex

01/21/2017

What is the MHGen Dex? It is a progress tracking mobile application for Capcom's Monster Hunter Generations (MHGen for short), built by a completionist, for completionists. In particular, it is built for Android 5.0 (Lollipop), and allows users to keep track of progress in their quest to collect all of the items strongest weapons and armors--all with the speed and convenience of a mobile application.

The MHGen Dex is capable of keeping track of both armor and weapon completion progress. Each tab displays a list of tiles, each displaying a compact set of information about the item, including it's name, rarity, hunter gender and hunter type.

As items are completed, the background color of the tile will change to indicate this. Items can be pinned and will appear first in the list for quick and easy access. Also, not only can specific items be searched for, but the list can also be filtered to only display items for your specific hunter setup--gender, hunter type and even specific weapon type!

Selecting any item on the list will bring up detailed information about it including stats, and crafting components. As these components are collected, they can be checked off, and will count towards their completion.

Kalibur Engine

01/21/2017

The Kalibur Engine is a text-based RPG development framework and engine, build in Java for use on Windows platforms. The engine allows a game to be dynamcially generated given a set of XML input files. Entire areas/location, weather systems, enemies/encounters, events, dialogue, items, and more can be generated from a set of files.

Provided with an XML file, the engine can dynamically generate an area's map, and populate it with "rooms", NPC's and events. Generated weather can effect which events occur, which NPC's are present, and can even provide Buffs and Debuffs in battle. Generated enemies can be encountered in battle, use generated attacks, and drop generated items. These items and gear can be equipped, upgraded, or sold to NPC's. And that's just to name a few

The highly customizable Events and Dialogue systems can trigger events based on nearly any condition in the game, ranging from date/time/location, to having a certain number of items, to having cleared a previous event, or having defeated an enemy. In fact, it is possible to generate complex systems such as gear crafting, or character and faction affinities, and their related events.
